# 🛡️ SafeWatch TN

SafeWatch TN is a real-time safety tracking and alert monitoring platform featuring a React + Vite frontend and a Node.js Express backend API with Socket.io web sockets.

## 📁 Project Structure

- **`backend/`**: Node.js Express API server with MongoDB integration and WebSockets for real-time alerting.
- **`frontend/`**: Modern React application powered by Vite, Tailwind CSS, Leaflet maps, and Recharts visualization.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js (v18 or higher)
- MongoDB (optional, falls back to Demo Mode automatically)

### 2. Running the Backend
```bash
cd backend
npm install
npm run dev
```
By default, the backend runs on port `5001`.

### 3. Running the Frontend
```bash
cd frontend
npm run dev
```
The frontend Vite server runs on `http://localhost:5173`.
